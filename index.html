Projects

Water Utility Mover App

This project was a collaboration between myself and four other team members. We created a web app for consumers to change their water utility service to a new location on a scheduled date. This app was designed using Flask, SQL, Python, and HTML. We used the USPS Address Validator API to verify the addresses the user entered were valid. A validator method was created to verify the user entered a valid future date.
